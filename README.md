India Car Sales Analysis Dashboard  2025

Project Overview

This project focuses on analyzing car sales data in India for the year 2025 using an interactive Power BI dashboard.
The dashboard provides deep insights into brand-wise performance, state-wise distribution, and top-selling car models.

Objective

The main goal of this project is to:

- Analyze sales performance of different car companies in India
- Identify top-performing car models
- Understand regional (state-wise) sales trends
- Build an interactive and business-ready dashboard

Key Features

-  Multi-page Dashboard (Brand-wise analysis)
-  Monthly Sales Trend Analysis
-  State-wise Sales Map (India)
-  Top Selling Car Model Identification
-  KPI Cards (Total Sales, GST,Discount)

Tools & Technologies

- Power BI (Data Visualization)
- Excel/csv  (Dataset)

Project Files

- "car dashboard.pbix"  Power BI Dashboard
- "car_sales_2025_india_updated.csv"  Dataset (India 2025)
- "images/"  Dashboard screenshots

Key Insights

- Certain car models dominate sales across multiple states
- Sales distribution varies significantly across regions in India
- Brand-wise comparison highlights market competition
- Monthly trends show fluctuations in demand


Unique Aspect of This Project

This dashboard uses a multi-page design approach, where each car brand has its own dedicated analysis page.
This improves clarity, usability, and provides a real-world business reporting experience.
